Notes

Usage / Security
  1. 上傳朋友的 Clash/Mihomo YAML。
  2. 系統會刪除原 proxies 節點。
  3. 系統會保留 rules、rule-providers、dns、proxy-groups 等設定。
  4. 輸入你的 vmess/vless 節點。
  5. 系統會自動新增國旗、國家策略組、通用策略組。
  6. 下載生成後的 YAML 並匯入 Clash/Mihomo。

LOG SAFE / 不記錄完整 vmess/vless 連結。

FILE MODE / 上傳、輸出、備份檔案以 600 權限儲存。
